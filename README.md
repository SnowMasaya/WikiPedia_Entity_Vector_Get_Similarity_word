### Install

If you don't install pyenv and virtualenv you have to install bellow
####Prepare Install
linux
```
apt-get install pyenv
apt-get install virtualenv
```
Mac
```
brew install pyenv
brew install virtualenv
```

####Prepare Inastall2
```
pyenv install 3.5.0
pyenv rehash
pyenv local 3.5.0
```

```
pip install -r requirement.txt
```
